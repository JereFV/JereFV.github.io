SEO

SEO significa Search Engine Optimization (Optimización para motores de búsqueda). Se trata del conjunto de estrategias y técnicas de optimización que se hacen en una página web para que aparezca orgánicamente en buscadores de Internet como Google, Yahoo o Youtube. La correcta aplicación del SEO puede causar incrementos expresivos en el tráfico y la visibilidad de las marcas en Internet.

¿Como hacer una web optimizada para SEO?

  1. Accesibilidad

    2. El primer paso a la hora de optimizar el SEO de una web es permitir el acceso a los motores de búsqueda a nuestro contenido. Es decir, hay que comprobar si la web es visible a los ojos de los motores de búsqueda y sobre todo, cómo están viendo la página. Por diversos motivos que explicaremos más adelante puede darse el caso de que los motores de búsqueda no puedan leer correctamente una web, un requisito indispensable para el posicionamiento.

  2. Indexabilidad

    3. Una vez que el robot de los navegadores acceden a una página el siguiente paso es que la indexe, estas páginas se incluirán en un índice donde se ordenan según su contenido, su autoridad y su relevancia para que sea más sencillo y rápido para el navegador acceder a ellas.
    El contenido duplicado es uno de los principales problemas para conseguir una correcta indexabilidad. Tener contenido duplicado significa que para varias URLs tenemos el mismo contenido. Este es un problema muy común, que muchas veces es involuntario y que además puede acarrear efectos negativos en el posicionamiento en el buscador, una redirección en el servidor para asegurarte de que solo hay una página que se muestre a los usuarios, es una solución efectiva.

  3. Contenido

    4. El contenido es la parte más importante de una web y por mucho que esté bien optimizada a nivel SEO, si no es relevante con respecto a las búsquedas que realizan los usuarios nunca va a aparecer en las primeras posiciones.
    Para hacer un buen análisis del contenido de nuestra web tienes unas cuantas herramientas a tu disposición, pero al final lo más útil es usar la página con el Javascript y los CSS desactivados cómo explicamos anteriormente. De este modo verás qué contenido está leyendo el navegador realmente y en qué orden está dispuesto.

Meta etiquetas

Las meta tags, o etiquetas, son fragmentos de código claves que aparecen en el código HTML de una web y que ayudan a los motores de búsqueda a interpretar mejor la página ofreciéndole información extra de la misma.

Algunas de las principales meta etiquetas están contenidas en la siguiente tabla:

Nombre etiqueta Descripción
Title La etiqueta title es la encargada de decirle a los motores de búsqueda la idea o tema principal sobre el que trata el contenido de la página que va a rastrear.
Description Complemento a la etiqueta Title, con la meta description le daremos una información algo más extensa a los motores de búsqueda del contenido de nuestra página. Trataremos de detallar con la mayor precisión posible el contenido que vamos a ofrecer al usuario.
Etiquetas Hs: Jerarquía de encabezados Como su nombre indica, los encabezados servirán para jerarquizar y ordenar el contenido de una página, por orden de importancia (h1, h2, h3…).
Canonical La meta etiqueta canonical es utilizada para decir al navegador cual es la página original en el caso de encontrarnos con páginas similares o idénticas.
ALT A la hora de posicionarnos en el buscador, no debemos olvidar el posicionamiento en las secciones de imagénes. Para esto, haremos uso de la etiqueta ALT (texto alternativo), la cual ayudará al navegador a saber de qué trata nuestra imagen.
Graph Las etiquetas Open Graph, son un protocolo que nos permite indicar a las redes sociales que información mostrar cada vez que se comparta un enlace de nuestro sitio web.

Relación de Meta etiquetas y SEO

Estas, son elementos relevantes para la estrategia de SEO de cualquier empresa, ayudan en el posicionamiento y, en consecuencia, en el flujo de visitantes para el sitio web. Los meta tags tienen dos funciones: contribuir al posicionamiento orgánico de una página y servir de primera impresión para los usuarios que hagan una búsqueda en algún navegador de internet.
Por lo que en conclusión, el propósito del SEO no sería posible de cumplirse sin el uso de las meta etiquetas y especialmente, la adecuada elección de las mismas.
